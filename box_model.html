Boks modellen

Boksmodellen består av innholdet, padding , border og margin. Den er formet som en boks.

Margin

margin er luft rundt innholdet. denne luften er usynlig.

Border

Border-egenskapen i CSS lar deg sette en ramme eller kant rundt et element. Hvor du kan endre stil, farge og størelse på rammen.

Selector i CSS

En Selector i CSS er del av et regelsett som faktisk velger området du vil forandre på. Eks: "Color: green" eller "font-size"

Attribute i CSS

Attribute selector i CSS velger innhold med like egenskaper og gruperer dem. eks: å forandre på bakgrunnsfargen på flere boks modeller

Value i CSS

Value i CSS endrer egenskapene til innhold i CSS. eks "red, blue"

"P" tag

man bruker "P" taggen for å skape paragrafer, som er en nettsides hovedtekst/brødtekst.

"H1" tag

"H1" taggen definerer den viktigste overteksten på nettsiden. "denne kan følges av "H2" og "H3"

padding

Paddingen er den innerste delen av boksmodellen, som også er området rundt tekst/elementet i boksen

Hvordan kommentere på Html

Man kommenterer på HTML ved bruk av kommandoen "< !-- -->"

Hvordan kommentere på CSS

Man kommenterer på CSS ved bruk av kommandoen */ og /*

Hvordan skriver man lenker

Man bruker kommandoen "a", og i href linjen skriver man nettsidens navn.

Hvordan legger man til bilder

Man legger til bilder ved å skrive "img", og så skriver man filnavnet som man har i samme mappe til bildet i src (source) linjen.

Hvordan legger man til bakgrunnsfarge

Man kan legge "background-color" og en value på ulike sekjsoner i en main eller i en div, for å endre bakgrunnen.

Hva brukes main i Html til

Main gjelder for hele dokumentet, og kan f:eks forandre størrelse og plassering på alle objekter. kan ikke være mere enn 1 main per dokument.

Hvordan skriver man farge med HEX-verdi

Man skriver nøyaktig farge med hexverdi, da det er nøyaktig fargekode. "background-color: #000666"

Hvordan skriver man farge med RGB-verdi

RGB-verdi er bare verdien på fargene rød, grønn og blå Eks: {background-color:rgb(255,0,0);} /* rød / {background-color:rgb(0,255,0);} / grønn / {background-color:rgb(0,0,255);} / blå */

Hva brukes flexbox i CSS til

Flexbox brukes til å ordne layout og plassere innhold i main.
